Full API Reference
==================

The full, machine-generated API listing is produced by Doxygen. After running ``make html`` (or ``make doxygen``), open the generated index at ``_build/doxygen-html/index.html`` to browse every function, struct, and enum from ``mir/mir.h``, ``mir/mir-gen.h``, and ``mir/c2mir/c2mir.h``.
